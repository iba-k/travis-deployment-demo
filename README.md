## Travis CIを使用した`gh-pages`へのデプロイのサンプル

`master`ブランチで変更があると以下のタスクを[Travis CI](https://travis-ci.org/)上で自動的に実行して`gh-pages`にデプロイします。

* CSVファイルをJSONに変更
* JSONが問題なく生成されているかを確認
* `gh-pages`ブランチに`push`
